## 4. 核心算法设计

### 4.1 中文分词算法

本系统实现了基于**二元组（Bigram）的中文分词算法**，将连续的中文文本切分为二元词组。

#### 4.1.1 算法原理

中文分词采用逐字符扫描的方式：

1. **字符类型判断**：判断当前字符是中文、英文还是数字
2. **二元组构建**：对连续的中文字符，优先取相邻两个字符组成二元组
3. **英文词提取**：对连续的英文和数字字符，提取长度 ≥ 2 的词
4. **停用词过滤**：移除无实际意义的虚词和学术模板词

#### 4.1.2 停用词表设计

停用词表包含三类词：

| 类别 | 示例 | 说明 |
|-----|------|------|
| 中文虚词 | 的、了、在、是、我、都 | 无实际语义 |
| 英文虚词 | the、is、are、was、were | 无实际语义 |
| 学术模板词 | 摘要、关键词、引言、结论 | 论文格式性文字 |

#### 4.1.3 算法复杂度

| 指标 | 复杂度 |
|-----|--------|
| 时间复杂度 | O(n)，n 为文本长度 |
| 空间复杂度 | O(m)，m 为分词结果数量 |

### 4.2 TF-IDF 算法

**TF-IDF（词频-逆文档频率）** 是信息检索领域的经典算法，用于评估词语对文档的重要程度。

#### 4.2.1 TF（词频）计算

$$TF(t, d) = \frac{f(t, d)}{\sum_{t' \in d} f(t', d)}$$

其中 $f(t, d)$ 为词 $t$ 在文档 $d$ 中的出现次数。

#### 4.2.2 IDF（逆文档频率）计算

$$IDF(t) = \ln\frac{N + 1}{df(t) + 1} + 1$$

其中 $N$ 为文档总数，$df(t)$ 为包含词 $t$ 的文档数。加 1 为平滑处理，避免除零。

#### 4.2.3 TF-IDF 向量化

$$TFIDF(t, d) = TF(t, d) \times IDF(t)$$

每篇文档被表示为一个 TF-IDF 向量，维度为词汇表大小。

#### 4.2.4 算法实现

```javascript
// TFIDFCalculator 类
class TFIDFCalculator {
  addDocument(id, text)  // 添加文档并计算词频
  build()                // 构建词汇表和 IDF
  getVector(id)          // 获取文档的 TF-IDF 向量
  getTopTerms(id, n)     // 获取文档的 Top-N 关键词
}
```

### 4.3 余弦相似度算法

**余弦相似度** 通过计算两个向量的夹角余弦值来衡量其相似程度。

#### 4.3.1 计算公式

$$\cos(\vec{a}, \vec{b}) = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}| \times |\vec{b}|} = \frac{\sum_{i=1}^{n} a_i \times b_i}{\sqrt{\sum_{i=1}^{n} a_i^2} \times \sqrt{\sum_{i=1}^{n} b_i^2}}$$

#### 4.3.2 相似度等级划分

| 相似度范围 | 等级 | 说明 |
|-----------|------|------|
| ≥ 80% | 高度相似 | 极有可能存在抄袭 |
| 50% - 79% | 中度相似 | 部分内容相似 |
| 20% - 49% | 低度相似 | 少量内容重合 |
| < 20% | 基本不相似 | 内容独立 |

#### 4.3.3 算法特性

| 特性 | 说明 |
|-----|------|
| 值域 | [0, 1]，0 表示完全不相似，1 表示完全相同 |
| 对称性 | sim(A, B) = sim(B, A) |
| 零向量处理 | 零向量与任何向量的相似度为 0 |
| 长度无关 | 只关注方向，不受文档长度影响 |

## 附录 A：算法核心代码

### A.1 中文分词核心代码

```javascript
export function tokenize(text) {
  if (!text || typeof text !== 'string') return [];

  const normalized = text.replace(/[\r\n]+/g, ' ').replace(/\s+/g, ' ').trim().toLowerCase();
  const tokens = [];
  let i = 0;

  while (i < normalized.length) {
    const char = normalized[i];

    if (isChinese(char)) {
      // 中文字符：优先取二元组
      if (i + 1 < normalized.length && isChinese(normalized[i + 1])) {
        tokens.push(char + normalized[i + 1]);
        i += 2;
      } else {
        tokens.push(char);
        i++;
      }
    } else if (isEnglish(char) || isDigit(char)) {
      // 英文和数字：提取完整单词
      let word = '';
      while (i < normalized.length && /[a-zA-Z0-9\-_]/.test(normalized[i])) {
        word += normalized[i]; i++;
      }
      if (word.length >= 2) tokens.push(word);
    } else {
      i++;
    }
  }
  return tokens;
}
```

### A.2 余弦相似度核心代码

```javascript
export function cosineSimilarity(vectorA, vectorB) {
  if (!vectorA || !vectorB) return 0;
  if (vectorA.length !== vectorB.length) return 0;

  let dotProduct = 0, normA = 0, normB = 0;
  for (let i = 0; i < vectorA.length; i++) {
    dotProduct += vectorA[i] * vectorB[i];
    normA += vectorA[i] * vectorA[i];
    normB += vectorB[i] * vectorB[i];
  }

  normA = Math.sqrt(normA);
  normB = Math.sqrt(normB);
  if (normA === 0 || normB === 0) return 0;

  return dotProduct / (normA * normB);
}
```
