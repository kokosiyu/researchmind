## 一、应用概述

### 1.1 什么是智研笔记？

**智研笔记（ResearchMind）** 是一款专为科研人员、研究生和学术工作者打造的智能论文管理与笔记协作平台。

### 1.2 核心功能

| 功能模块 | 功能描述 |
|---------|---------|
| 📚 论文管理 | 上传、解析、管理学术论文，支持PDF/TXT/DOCX格式 |
| 🤖 AI分析 | 自动提取论文信息、生成摘要、关键词 |
| 📝 笔记功能 | Markdown格式笔记，支持图片上传 |
| 🔗 知识图谱 | 可视化展示论文、作者、关键词关系 |
| 💬 AI助手 | 智能问答、论文解读、写作辅助 |
| 🔐 数据隔离 | 每个用户拥有独立的论文库 |

### 1.3 适用人群

- **研究生新生**：快速建立个人论文库
- **科研人员**：管理项目相关文献
- **学术写作者**：论文润色与辅助写作
- **教授/导师**：追踪领域最新研究

## 二、快速开始

### 2.1 访问应用

- **本地访问**：打开浏览器访问 `http://localhost:5000`
- **公网访问**：通过 cpolar 提供的公网地址访问

### 2.2 注册与登录

#### 第一步：注册账户

1. 点击首页右上角「立即注册」按钮
2. 填写注册信息：
   - **用户名**：设置你的登录账号（唯一）
   - **邮箱**：用于找回密码（唯一）
   - **密码**：设置登录密码
3. 点击「注册」按钮完成注册

#### 第二步：登录系统

1. 点击首页右上角「立即登录」按钮
2. 输入用户名和密码
3. 点击「登录」按钮进入系统

#### 第三步：开始使用

登录成功后，你将进入**数据看板**页面，此时你的论文库是空的，需要上传论文开始使用。

## 三、功能模块详解

### 3.1 数据看板（Dashboard）

**功能定位**：总览个人论文库概览

**页面元素说明**：

| 区域 | 说明 |
|-----|------|
| 统计卡片 | 显示论文总数、笔记总数、本月新增论文数 |
| 年度分布 | 柱状图展示各年份论文数量 |
| 研究领域漏斗 | 可视化展示研究主题分布 |
| 最近论文列表 | 快速访问最近上传的论文 |
| 搜索框 | 快速搜索论文标题 |

### 3.2 论文上传与分析（Analyze）

**功能定位**：上传并解析学术论文

**操作步骤**：

#### 方式一：拖拽上传

1. 进入「论文分析」页面
2. 将 PDF/TXT/DOCX 文件拖拽到上传区域
3. 等待文件上传和 AI 分析完成
4. 查看 AI 提取的信息并确认修改
5. 点击「保存到论文库」完成

#### 方式二：点击上传

1. 点击上传区域的「选择文件」按钮
2. 选择本地论文文件
3. 后续步骤同拖拽上传

**AI 分析功能**：

- ✅ **自动提取标题**：从论文内容智能识别标题
- ✅ **提取作者信息**：识别论文作者
- ✅ **生成摘要**：AI 自动生成论文摘要
- ✅ **提取关键词**：智能识别核心关键词

**支持的文件格式**：

| 格式 | 说明 |
|-----|------|
| PDF | 学术论文常用格式（支持扫描件OCR） |
| TXT | 纯文本文件 |
| DOCX | Microsoft Word 文档 |

**注意事项**：

- 单个文件大小限制：**50MB**
- 加密 PDF 需要先解密后上传
- 建议上传清晰可识别的论文文件

### 3.3 知识图谱（Graph）

**功能定位**：可视化展示论文、作者、关键词之间的关系

**页面元素说明**：

| 元素 | 说明 |
|-----|------|
| 力导向图 | 节点代表论文/作者/关键词，边代表关系 |
| 节点类型 | 📄论文（蓝色）、👤作者（绿色）、🏷️关键词（橙色） |
| 边类型 | 引用关系、合著关系、主题关联 |
| 图例 | 右上角显示节点和边的含义 |

**交互操作**：

| 操作 | 效果 |
|-----|------|
| 点击节点 | 显示节点详情信息 |
| 拖拽节点 | 移动节点位置 |
| 鼠标滚轮 | 缩放视图 |
| 拖拽空白区域 | 平移视图 |

### 3.4 工作台（Workbench）

**功能定位**：论文管理与笔记编写的核心工作区

**左侧：论文列表**

- 显示所有已上传的论文
- 支持按标题搜索过滤
- 点击论文查看详情
- 支持删除论文（需确认）

**右侧：笔记编辑**

- **Markdown 编辑器**：支持丰富的 Markdown 语法
- **实时预览**：编辑内容实时渲染预览
- **图片上传**：支持粘贴图片到编辑器
- **标签管理**：为笔记添加标签方便分类

**笔记操作**：

| 操作 | 说明 |
|-----|------|
| 创建笔记 | 点击「+ 添加笔记」按钮 |
| 保存笔记 | 自动保存，无需手动保存 |
| 删除笔记 | 点击笔记旁的删除图标 |
| 添加标签 | 在笔记编辑区域输入标签 |

### 3.5 搜索功能（Search）

**功能定位**：快速检索论文库中的内容

**搜索方式**：

| 方式 | 说明 |
|-----|------|
| 标题搜索 | 输入关键词搜索论文标题 |
| 作者搜索 | 输入作者名字搜索 |
| 关键词搜索 | 搜索论文的关键词 |
| 全文搜索 | 搜索论文内容和摘要 |

**搜索结果**：

- 显示匹配的论文列表
- 高亮显示匹配关键词
- 支持点击查看详情

### 3.6 个人中心（Profile）

**功能定位**：管理个人账号信息

**可修改项**：

| 项目 | 说明 |
|-----|------|
| 用户名 | 修改登录用户名 |
| 邮箱 | 修改绑定邮箱 |
| 删除账户 | 永久删除个人账号及所有数据 |

**注意事项**：

- 删除账户后**无法恢复**所有数据
- 修改邮箱后需要重新登录

## 四、AI智能助手使用指南

### 4.1 打开助手

点击页面右下角的 🤖 图标，展开 AI 助手聊天窗口。

### 4.2 功能介绍

**支持的功能**：

| 功能类别 | 示例问题 |
|---------|---------|
| 论文解读 | "帮我总结这篇论文的核心观点" |
| 写作辅助 | "帮我润色这段文字" |
| 研究建议 | "如何提高研究效率？" |
| 方法推荐 | "推荐一些研究方法" |

### 4.3 快捷问题

点击快捷问题按钮快速提问：
- 帮我总结这篇论文的核心观点
- 如何提高研究效率？
- 帮我润色这段文字
- 推荐一些研究方法

### 4.4 使用技巧

**提问技巧**：

1. **明确问题**：提问时尽量具体
2. **上下文**：可以引用当前论文内容
3. **多轮对话**：支持连续追问
4. **复制粘贴**：可以粘贴论文段落进行分析

**注意事项**：

- 单个消息长度限制：**2000字**
- 提问频率限制：**30次/分钟**
- 响应时间：根据问题复杂度可能需要几秒到几十秒

## 五、常见问题解答

### 5.1 注册登录问题

**Q：注册失败怎么办？**

A：请检查：
- 用户名是否已被注册
- 邮箱格式是否正确
- 密码是否符合要求

**Q：忘记密码怎么办？**

A：目前暂不支持密码找回功能，请联系管理员重置密码。后续版本将添加此功能。

**Q：为什么登录后看不到论文？**

A：每个用户拥有独立的论文库，登录后需要自己上传论文。如果之前能看到别人的论文，说明数据隔离功能已修复，现在是正常现象。

### 5.2 论文上传问题

**Q：上传文件失败？**

A：请检查：
- 文件格式是否支持（PDF/TXT/DOCX）
- 文件大小是否超过 50MB
- 文件是否损坏或加密

**Q：AI 分析时间太长？**

A：论文分析需要调用 AI 服务，复杂论文可能需要较长时间，请耐心等待。

**Q：上传的论文在哪里？**

A：论文保存在你的个人论文库中，可以在「工作台」或「数据看板」查看。

### 5.3 笔记功能问题

**Q：笔记如何关联论文？**

A：在工作台左侧选中一篇论文后，创建的笔记会自动关联到该论文。

**Q：如何在笔记中插入图片？**

A：直接复制图片后粘贴到编辑器中，图片会自动上传。

**Q：笔记支持什么格式？**

A：支持 Markdown 格式，包括标题、列表、链接、图片、代码块等。

### 5.4 知识图谱问题

**Q：图谱显示空白？**

A：需要至少上传几篇论文才能生成图谱关系。

**Q：如何查看节点详情？**

A：点击图谱中的节点即可查看详情信息。

### 5.5 AI助手问题

**Q：AI 助手请求失败？**

A：可能原因：
- 网络连接问题
- AI 服务暂时不可用
- 请求过于频繁（超过 30 次/分钟）

**Q：AI 回答不准确？**

A：可以尝试更明确地描述问题，或提供更多上下文信息。
