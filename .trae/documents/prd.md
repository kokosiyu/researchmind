
## 1. Product Overview
科研论文智能分析与知识图谱平台 - 帮助研究人员高效管理、分析和发现学术文献之间的关联关系
- 针对科研工作者、学生和学者，解决文献管理困难、研究脉络不清晰、发现新文献效率低等问题
- 目标是成为科研工作的得力助手，提升研究效率，促进学术创新

## 2. Core Features

### 2.1 User Roles
| Role | Registration Method | Core Permissions |
|------|---------------------|------------------|
| Researcher | 邮箱注册/演示模式 | 上传论文、构建知识图谱、分析文献关联、保存研究成果 |

### 2.2 Feature Module
1. **首页**：英雄区、功能介绍、论文上传入口、示例演示
2. **论文分析页**：论文元数据展示、摘要提取、关键词分析、引用关系可视化
3. **知识图谱页**：交互式知识图谱、文献关联探索、聚类分析、时间线展示
4. **研究工作台**：文献库管理、标签系统、笔记功能、导出报告

### 2.3 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 首页 | 英雄区 | 动态3D背景、引人入胜的标题、CTA按钮、渐入动画 |
| 首页 | 功能介绍 | 卡片式布局、图标动画、悬停效果、滚动触发 |
| 论文分析页 | 元数据面板 | 论文标题、作者、期刊、年份、DOI等信息展示 |
| 论文分析页 | 文本分析 | 自动摘要生成、关键词提取、研究领域分类 |
| 知识图谱页 | 交互式图谱 | 力导向图布局、节点缩放拖拽、关联高亮、详情弹窗 |
| 知识图谱页 | 控制面板 | 筛选功能、时间范围、研究领域、布局切换 |
| 研究工作台 | 文献库 | 网格/列表视图切换、排序、搜索、批量操作 |
| 研究工作台 | 笔记系统 | 富文本编辑、引用标注、标签管理 |

## 3. Core Process
用户上传论文 → 系统自动解析提取信息 → 构建知识图谱 → 用户交互式探索 → 保存和导出研究成果

```mermaid
flowchart LR
    A[用户访问首页] --> B[上传论文]
    B --> C[解析论文内容]
    C --> D[提取元数据与关键词]
    D --> E[构建知识图谱]
    E --> F[交互式探索]
    F --> G[添加笔记与标签]
    G --> H[导出研究报告]
```

## 4. User Interface Design
### 4.1 Design Style
- **主色调**：深邃蓝 (#1e3a8a) + 科技紫 (#7c3aed) + 明亮青 (#06b6d4)
- **辅助色**：中性灰阶、白色背景
- **按钮风格**：圆角矩形、渐变背景、悬停发光效果
- **字体**：使用现代无衬线字体，标题采用更有设计感的字体
- **布局风格**：卡片式布局、多层次阴影、玻璃态效果
- **图标风格**：线性图标 + 填充图标结合，动态图标效果

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 首页 | 英雄区 | 全屏高度、渐变背景、3D粒子效果、打字机动画、大CTA按钮 |
| 首页 | 功能介绍 | 网格布局、彩色卡片、图标动画、悬停上浮效果 |
| 论文分析页 | 侧边栏 | 可折叠、背景模糊、平滑过渡 |
| 论文分析页 | 内容区 | 两栏布局、卡片容器、图表组件、滚动进度条 |
| 知识图谱页 | 画布 | 深色背景、动态连线、节点发光、缩放控制 |
| 研究工作台 | 工具栏 | 固定顶部、搜索框、筛选按钮、批量操作 |

### 4.3 Responsiveness
- Desktop-first设计，响应式适配平板和移动设备
- 触摸优化：更大的点击区域、滑动手势支持
- 知识图谱在移动端简化为列表视图

### 4.4 3D Scene Guidance
- 首页英雄区：3D粒子系统模拟知识流动，鼠标交互影响粒子运动
- 知识图谱：可选3D视图模式，使用Three.js渲染，支持旋转和缩放
- 环境：深色星空背景，蓝色和紫色光效
- 光照：柔和的环境光 + 定向光源，营造科技感
- 交互：鼠标悬停高亮节点，点击显示详情
